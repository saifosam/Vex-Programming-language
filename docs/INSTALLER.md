Building a desktop installer for Vex (Windows)

This repository currently builds a Python package for `vex-lang`, but you can also
create a standalone Windows executable using PyInstaller.

Prerequisites

- Windows
- Python 3.11 installed
- `pip` available
- The repository checked out locally

Build steps

1. Install PyInstaller:

```powershell
python -m pip install --upgrade pyinstaller
```

2. Run the build script:

```powershell
scripts\build_installer.bat
```

If `assets\vex.ico` is present, PyInstaller will embed that icon into `dist\vex.exe` automatically.

What it produces

- `dist\vex.exe` — a standalone executable that includes the CLI entrypoint.
- `build\` — build files used by PyInstaller.
- `vex.spec` — spec file generated by PyInstaller.

How to use the executable

```powershell
.\dist\vex.exe examples\counter.vex
```

Packaging as a proper installer

If you want a real installer rather than a single executable, use a Windows installer
creator such as Inno Setup or NSIS. Those tools can wrap `dist\vex.exe` and create a
start menu shortcut, desktop icon, and uninstall entry.

Example Inno Setup steps:

1. Install Inno Setup from https://jrsoftware.org/isinfo.php.
2. Create a script that copies `dist\vex.exe`, `README.md`, and any example files.
3. Build the installer.

Notes

- This executable will run with the Python runtime bundled by PyInstaller.
- It is not a full OS-native installer yet; it is a first step toward a packaged app.
- For macOS or Linux installers, use PyInstaller or platform-specific packaging tools.
