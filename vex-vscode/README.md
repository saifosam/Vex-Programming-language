# Vex Language Extension

This VS Code extension adds support for the Vex DSL:

- `.vex` file association
- syntax highlighting for `style:`, `logic:`, and `view:` sections
- a custom Vex file icon when the Vex icon theme is enabled

## Install

Install the package from the VS Code Marketplace once published, or use `vsce package` to build it locally.

## Usage

Open any `.vex` file and select the language mode if it is not automatically detected.

## Publish

If you publish this extension, change the `publisher` field in `package.json` to your publisher ID.
